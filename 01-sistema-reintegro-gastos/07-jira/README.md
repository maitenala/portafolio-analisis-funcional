# Gestión de Proyecto en Jira

### Descripción

Este apartado tiene como objetivo mostrar una simulación de gestión de requerimientos utilizando Jira como herramienta de seguimiento y representar el trabajo de un equipo ágil.

## Configuración realizada

- Creación del backlog inicial con historias de usuario y tareas.
- Organización de los requerimientos en sprints.
- Configuración del tablero de seguimiento.
- Creación y gestión de bugs asociados a funcionalidades.
- Seguimiento del estado de las tareas e incidencias.
